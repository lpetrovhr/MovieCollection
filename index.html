MovieCollection

Welcome to my first node.js express app. This is little movie collection app when you can add movie you watched and users can see your collection.

Technologies I used in development:

  • Node.js(Express framework)
  • NoSQL Database(mongo)
  • Gulp(jshint, jscs, istanbul, mocha)
  • Heroku(deployment)
  • Google Authentication
  • Github

You can visit deployed application here: https://awesome-movie-collection.herokuapp.com/
